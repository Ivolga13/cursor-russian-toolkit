# Изучение React

## Что такое React
**React** — JavaScript-библиотека для создания пользовательских интерфейсов, разработанная компанией **Facebook**.

## Основные концепции

### Компоненты
Переиспользуемые блоки кода, которые описывают части интерфейса.

### Props
Свойства, которые передаются в компоненты для настройки их поведения.

### State
Внутреннее состояние компонента, которое может изменяться со временем.

## JSX — расширение синтаксиса
**JSX** позволяет писать HTML-подобный код внутри JavaScript, делая код более читаемым и интуитивным.

## Hooks — современный подход
**Hooks** появились в **React 16.8** и позволяют использовать состояние и другие возможности React в функциональных компонентах.
