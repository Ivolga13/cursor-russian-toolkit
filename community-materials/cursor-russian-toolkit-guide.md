# Cursor Russian Toolkit — Полный набор для русскоязычных пользователей

*Профессиональная система Skills и Rules для работы с русскими текстами в Cursor*

## 🎯 Что это такое?

**Cursor Russian Toolkit** — это комплексная система инструментов, которая превращает Cursor в мощный редактор для русскоязычного контента. Система включает специализированные Rules и Skills, которые:

- ✅ **Автоматически исправляют** русскую типографику
- ✅ **Улучшают стиль** письма и структуру документов
- ✅ **Работают со смешанным контентом** (русский + английский)
- ✅ **Автоматизируют** процесс создания качественных документов
- ✅ **Экономят до 85% времени** на создание профессиональных материалов

## 🚀 Быстрый старт (5 минут)

### Что вы получите
После установки у вас будет:
- Автоматическая коррекция русской типографики
- Специализированные помощники для улучшения текстов
- Система автоматизации для создания документов
- Готовые шаблоны для разных типов контента

### Установка
1. Скачайте файлы из этого репозитория
2. Скопируйте папки `.cursor/rules/` и `.cursor/skills/` в ваш проект
3. Перезапустите Cursor
4. Готово! Система активна

## 📋 Что входит в комплект

### Rules (Автоматические правила)
- **russian-typography.mdc** — Русская типографика (кавычки, тире, пробелы)
- **russian-writing-style.mdc** — Стиль русского письма
- **mixed-language-content.mdc** — Смешанный русско-английский контент
- **workflow-automation.mdc** — Автоматизация рабочих процессов

### Skills (Специализированные помощники)
- **russian-text-enhancer** — Комплексное улучшение русских текстов
- **auto-processor** — Полная автоматизация обработки документов
- **writing-assistant** — Улучшение стиля и читаемости
- **content-organizer** — Структурирование и организация контента
