## 🔧 Кастомизация для организаций

### Корпоративные настройки
Создайте файл `.cursor/corporate-standards.md`:

```markdown
# Корпоративные стандарты [Название компании]

## Стиль документов
- Тон: профессиональный, но дружелюбный
- Структура: обязательные разделы "Цель", "Результат", "Следующие шаги"
- Терминология: используем корпоративный глоссарий

## Автоматизация
- Все документы проходят через Auto Processor Full Mode
- Презентации обязательно в Presentation Mode
- Техническая документация требует примеров кода

## Качество
- Обязательная проверка Russian Text Enhancer
- Все файлы следуют File Naming Rule
- Консистентная типографика во всех материалах
```

### Образовательные учреждения
Адаптируйте Rules для учебных задач:

```markdown
# Настройки для учебных заметок

## Обязательные разделы
- "Что я изучил(а)" — ключевые концепции
- "Вопросы" — что хочется понять лучше  
- "Практика" — что попробовал(а) на практике
- "Следующие шаги" — планы на изучение

## Стиль
- Личные размышления приветствуются
- Неформальный тон допустим
- Обязательные примеры и иллюстрации
```
