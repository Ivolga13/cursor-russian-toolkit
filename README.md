## 🎯 Что это такое?

**Cursor Russian Toolkit** — это набор специализированных инструментов, который превращает Cursor в мощный редактор для работы с русскоязычным контентом. Система автоматически исправляет типографику, улучшает стиль письма и помогает создавать профессиональные документы за считанные минуты.

### ⚡ Быстрые результаты
- **85% экономии времени** на создание документов
- **Автоматическое исправление** русской типографики
- **Профессиональное качество** каждый раз
- **Готовые презентации** за 3 минуты

## 🚀 Быстрый старт

### 1. Установка (2 минуты)
```bash
# Скачайте репозиторий
git clone https://github.com/your-username/cursor-russian-toolkit.git

# Скопируйте файлы в ваш проект
cp -r cursor-russian-toolkit/.cursor/ your-project/
```

### 2. Первое использование
1. Создайте файл `test.md`
2. Напишите любой русский текст
3. Попросите: `"Используй Russian Text Enhancer"`
4. Наслаждайтесь результатом! 🎉

## 📦 Что входит в комплект

### 🔧 Rules (Автоматические правила)
- **russian-typography** — Кавычки, тире, пробелы
- **russian-writing-style** — Стиль и структура текста
- **mixed-language-content** — Русско-английский контент
- **workflow-automation** — Автоматизация процессов

### 🤖 Skills (AI-помощники)
- **russian-text-enhancer** — Комплексное улучшение русских текстов
- **auto-processor** — Полная автоматизация обработки
- **writing-assistant** — Улучшение стиля и читаемости
- **content-organizer** — Структурирование контента
