# Writing Assistant

Helps improve the clarity, grammar, and style of your notes and documentation.

## When to Use

- Reviewing notes for clarity and readability
- Improving sentence structure and flow
- Checking grammar and spelling
- Making text more engaging and accessible
- Ensuring consistent tone and style

## Review Process

### 1. Clarity Check
- Are sentences clear and easy to understand?
- Is the main point obvious in each paragraph?
- Are there any confusing or ambiguous phrases?

### 2. Structure Review
- Does the content flow logically?
- Are paragraphs focused on single ideas?
- Are headings descriptive and helpful?

### 3. Style Improvements
- Use active voice when possible
- Keep sentences under 25 words
- Choose simple words over complex ones
- Remove unnecessary words and phrases

### 4. Grammar and Mechanics
- Check for spelling errors
- Ensure proper punctuation
- Verify consistent tense usage
- Fix run-on sentences

## Feedback Format

Provide suggestions in this format:

**Original:**
```
The thing that I learned today was about how to use Cursor skills which are really helpful for making your work better.
```

**Improved:**
```
Today I learned about Cursor skills. These tools help improve your workflow and productivity.
```

**Why it's better:**
- Shorter, clearer sentences
- Removed unnecessary words ("the thing that", "which are really")
- More direct and engaging

## Common Improvements

### Sentence Length
- ❌ Long: "The documentation that we created for this project includes all the information that new users need to understand how to get started with using the system effectively."
- ✅ Better: "Our documentation helps new users get started quickly. It includes setup instructions and basic usage examples."

### Active Voice
- ❌ Passive: "Mistakes were made in the code review process."
- ✅ Active: "We made mistakes during the code review."

### Simple Language
- ❌ Complex: "Utilize this methodology to facilitate optimization."
- ✅ Simple: "Use this method to improve performance."

## Tone Guidelines

For learning notes, maintain:
- **Conversational**: Write like you're explaining to a friend
- **Encouraging**: Focus on progress and learning
- **Clear**: Avoid jargon unless necessary
- **Personal**: Use "I learned" instead of "It was learned"
