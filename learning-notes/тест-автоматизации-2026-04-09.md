## Что такое Docker

Сегодня я начала изучать **Docker** — технологию для «контейнеризации» приложений. Это инструмент, который позволяет упаковывать приложения со всеми их зависимостями в легковесные, переносимые контейнеры.

## Основные команды

В ходе изучения освоила базовые команды **Docker**:

### Запуск контейнеров
```bash
docker run [опции] [образ]
```
Запускает новый контейнер из указанного образа.

### Сборка образов
```bash
docker build [опции] [путь]
```
Создает новый образ на основе **Dockerfile**.

### Просмотр контейнеров
```bash
docker ps
```
Показывает список запущенных контейнеров.

## Дополнительные инструменты

### Docker Compose
Изучила **Docker Compose** — инструмент для управления несколькими контейнерами одновременно. Особенно полезен для сложных приложений, состоящих из нескольких сервисов.
