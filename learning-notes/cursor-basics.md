# Cursor Basics - My First Learning Session

## What I Learned

Today I discovered that Cursor is more than just a code editor. It has special features called Skills and Rules that act like helpful assistants.

**Skills** are like having expert consultants available anytime. They help with specific tasks like improving writing or organizing notes.

**Rules** are like having a helpful teacher who automatically guides you toward good practices as you work.

## Important Details

### Skills I Can Use
- **Writing Assistant**: Helps make my notes clearer and easier to read
- **Content Organizer**: Helps structure my learning materials properly

### Rules That Help Me
- **Markdown Formatting**: Keeps my notes looking professional
- **File Naming**: Ensures I can always find my files easily

### How They Work Together
When I write notes, the Rules automatically check my formatting. Then I can use Skills to improve the content and organization.
